# ucla-takehome

Description:
This script pulls census data from an API endpoint, digests, and returns the data formatted as a CSV showing population and population change by year as well as a graphical representation of change in total state population year over year.
Requirements:
Python 3.X
Pip

Installation and Usage:
git clone this repository
cd into the repository
pip -r requirements.txt
python (or your chosen alias) main.py
